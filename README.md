# Block Model Compression Algorithm — Project Portfolio

This repository contains my personal project documentation from a software engineering project, completed as part of a team at the University of Adelaide.

## About the Project

The project was a gamified industry challenge set by **Maptek**, an Adelaide-based mining technology company. The goal was to develop a high-performance lossless compression algorithm for 3D block models — large geological data structures used in mining — and compete on a leaderboard scored on compression ratio and processing speed.

The final solution was built in **C++**, using a modular monolithic architecture with a producer-consumer threading model. Key techniques explored included RLE compression, a 26-ary tree structure for 3D neighbour lookup, multithreaded processing, and optimised input streaming.
